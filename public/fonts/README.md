# Fonts used for export (TTF/OTF)

Caption export uses FFmpeg/libass, which cannot load WOFF/WOFF2 webfonts. For that reason we keep TTFs here and prefer them during export.

- `inter.ttf`
- `montserrat.ttf`
- `dm-sans.ttf`
- `poppins-regular.ttf`, `poppins-bold.ttf`, `poppins-black.ttf`
- `noto-sans.ttf` (export fallback)
